# Sistema-locadora-jogos-digitais
Sistema de Banco de Dados para locadora de jogos digitais. O modelo lógico foi desenvolvido em DBML no dbdiagram.io, contemplando entidades Cliente, Jogo, Locação e Item_Locação, com definição de chaves primárias, estrangeiras e cardinalidades (1:N e N:N), garantindo integridade e normalização adequada.
